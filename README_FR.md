# Gestionnaire de Temps d'Étude

Une application PHP pour suivre le temps d'étude par catégories et matières. Génère des rapports PDF et supporte plusieurs langues.

## Fonctionnalités

- Suivre le temps d'étude par catégories et matières
- Système d'authentification des utilisateurs
- Génération de rapports PDF
- Support multilingue (français, anglais, allemand, italien)
- Récupération de mot de passe

## Prérequis

- PHP 7.4+ ou 8.x
- SQLite3
- Composer

## Installation

1. Cloner le dépôt
2. Installer les dépendances :
   ```bash
   composer install
   ```
3. Accéder à l'application via votre serveur web (ex: `http://localhost/public/`)

## Identifiants par défaut

- Nom d'utilisateur : `admin`
- Mot de passe : `admin123`

## Langues

Changez entre français, anglais, allemand et italien depuis le sélecteur de langue dans l'application.
